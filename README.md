# Implats_VR #

# HOW TO INSTALL #

## clone from github ##
    prerequisites ###
        1) install git

        2) install Git LFS
            a) download   
                windows - 
                    https://github.com/git-lfs/git-lfs/releases
                mac - use homebrew
                    $ brew update
                    $ brew install git-lfs
            b) open command line and enter "git lfs install"
    
    clone the repo: "https://github.com/Dylan-Whiteford/Implats_VR.git"
